# Solace Repo Snapshot

## Primary goal

Provide a fast, accurate view of the current file and folder structure of the Solace project.

## When to use

- start of a session
- after changes
- before editing
- when structure feels unclear

## What to inspect

### Root
- top-level folders
- config files

### Key directories
- /app
- /components
- /lib
- /public
- /styles

### App structure
- tool pages
- routes
- layout files

### Components
- shared UI components

### Lib
- utilities
- safety files

### Public
- assets
- backgrounds

## Output format

## Repo Snapshot

### Root
(list)

### App structure
(list)

### Components
(list)

### Lib
(list)

### Public
(list)

## Summary
(short summary)

## Rules

- Do not modify files
- Do not guess
- Only report what exists
